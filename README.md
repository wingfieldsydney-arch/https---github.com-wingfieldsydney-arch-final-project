SoundSpace: Spotify Music Explorer
This project is a music discovery web application using the Spotify Web API. Official Planning Document Website Title: SoundSpace: Spotify Music Explorer Team Members + Roles Sydney - Project coordination, content writing, and UI planning Daniel - Lead design & CSS styling (layout, visuals, responsiveness) Yaneily - API integration and data handling (fetching Spotify data) Unknown - JavaScript functionality & user interaction (event handling, dynamic features)

Topic Description For our final project website is a music discovery platform built using the Spotify Web API. The goal of the site is to allow users to explore music in a more interactive and personalized way. Instead of just listening to music, users will be able to search for artists, albums, and songs, and view detailed information about them. The site will display data such as artist names, album covers, song titles, genres, and previews (if available). Users will also be able to browse trending or popular music and discover new artists based on categories or search results. Overall, the website focuses on helping users explore music visually and interactively, making it easier to discover new content and learn more about their favorite artists. API Selection Primary API: https://developer.spotify.com/documentation/web-api/ The Spotify Web API returns data about: Artists Albums Tracks (songs) Playlists Genres ( It also allows searching and retrieving music-related metadata (like images, names, and previews) Alternative APIs (if needed): Last.fm API https://www.last.fm/api Returns artist info, music trends, tags, and user listening data Deezer API https://developers.deezer.com/api Returns songs, albums, artist data, and preview clips

Site Structure Plan We plan to build at least 5 main pages:

Home Page Introduction to the website Featured or trending music Navigation to other pages
Search Page Search bar for artists, albums, or songs Display results from the Spotify API Interactive results (click to view more details)
Artist Page Detailed information about a selected artist Artist image, genres, and top tracks
Album / Track Page Album details (cover, tracklist) Song previews (if available) Release information
